# nestJs 语法学习项目

## 描述

根据[NEST中文文档](https://docs.nestjs.cn/6/controllers) 内容学习 nest 语法的demo 项目

**********

## 安装依赖

```bash
$ npm install
```

## 运行应用

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## 测试

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
